Welcome

TaskTicker will help you to keep track of all your daily activities. Sign in with Google to get extra functionality.

Guest can:

  • Create tasks with timers.
  • Download CSV with timings.

Signed-in User can:

  • Keep track of the time even with the tab closed (the state recovers on opening tab).
  • Have the history of all past tasks.
  • Analytics: See charts of time spend on each task for selected date range.
  • User can connect to the data from any device. Changes in one place propagated in real-time to all active clients for the same user.
